¿Que és esta página?

Esta página es un proyecto hecho para la asignatura de tecnología, con el objetivo de hacer una compañía ficticia. Nosotros hicimos una compañía ficticia que se desarrolla en el ámbito de piratería de videojuegos.

¿Que recursos fueron utilizados?

Para el video y el audio se utilizó adobe premiere y una compilación de distintos recursos, como el video de trivago.
Para el diseño y desarrollo de la página web se utilizó Visual Studio Code (Editor de texto), código HTML, CSS y js propio, en conjunto con Bootstrap 4.
